Paleta (tokens CSS)

Neutros fríos de fondo, superficie blanca y acento teal; naranja y verde para estados de negocio (pendiente / completado), como en listas y tableros de los UI kits de referencia.

Estados de tarea (UX)

Misma jerarquía que en plantillas de dashboard: título primero, meta después, indicador de estado a la izquierda.

Informe trimestral

Reunión de planificación

COMPLETED — confirmación positiva

Endpoints útiles

La API REST sigue bajo /api/tasks. Ejemplos desde el mismo origen:

Salud

GET /health

Listar con filtro

GET /api/tasks?status=PENDING

Crear

POST /api/tasks + JSON body
